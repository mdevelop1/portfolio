# M_Develop – Portfolio

Stack: Next.js 14 + TypeScript + TailwindCSS + Framer Motion

## Development

1. Install deps
```bash
npm install
```
2. Run dev server
```bash
npm run dev
```
3. Open http://localhost:3000

## Deploy
- Recommended: Vercel – import repository, build command `next build`.

## Contact API
- `pages/api/contact.ts` – currently logs payload. Integrate email (e.g., Nodemailer via SMTP, or Resend) and add env vars.

## Assets
- Put your screenshots into `public/images/portfolio/`.
- Replace `public/images/logo.svg` with your own logo if desired.
